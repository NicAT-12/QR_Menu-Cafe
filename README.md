# QR Menú Café

Sistema de menú digital accesible mediante código QR, diseñado para cafeterías. Permite a los clientes escanear un código QR y acceder directamente a un menú web con bebidas y descripciones de forma rápida, moderna y sin contacto.

**Demo en vivo:** https://menu-cafe-pied.vercel.app/  
**Repositorio:** https://github.com/NicAT-12/QR_Menu-Cafe  

---

## Descripción del proyecto

QR Menú Café es una aplicación web simple que reemplaza el menú físico tradicional por una versión digital accesible desde un código QR.

El objetivo del proyecto es ofrecer una experiencia rápida, limpia y responsive para clientes de cafeterías, mejorando la accesibilidad del menú desde cualquier dispositivo móvil.

---

## Funcionalidades

- Visualización de menú de café en formato web
- Acceso directo mediante código QR
- Descripción breve de cada producto
- Interfaz simple y enfocada en mobile-first
- Diseño liviano y rápido de cargar

# Ejecutar en desarrollo
